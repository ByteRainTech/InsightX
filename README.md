# InsightX
用于大数据工程的便捷式工具

# 支持功能
> 👏 告别重加载，`模型挂载` 为您跳过训练代码 Deving 的模型重复加载。

|功能|受支持版本|
|--|--|
|模型挂载|`0.1`|
|检索模型结构|`0.1`|
|dtype转移|`0.1`|

# 部署
本项目分为 `/server/` 以及 `/client/` 两个部分，如您只需要API接口，仅部署 `/server/`即可。
